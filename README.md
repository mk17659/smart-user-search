# Smart User Search Page

## How to Run
1. Run: npm install
   npm run dev

## Features
- Case-insensitive search
- Role & status filters
- Toggle active/inactive users
- Derived count info
- Custom hook with useMemo for performance

## Decisions
- Used custom hook to keep logic outside JSX
- Used static data as required
- No external libraries

## About Project
This project was created as part of an internship assignment.
It demonstrates React basics like state management, filtering,
and conditional rendering using static data.
